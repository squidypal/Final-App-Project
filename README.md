# Merge 2048

## Features

- **Swipe controls** for touch (and mouse-drag / arrow-keys / WASD in the Editor), with input locked during animations to prevent double moves.
- **Score & best score**, persisted between sessions.
- **One-step Undo** of the last move.
- **Win / continue** - reaching 2048 shows a win screen; "Keep Going" lets you chase a bigger tile.
- **Game-over detection** 
- **Lifetime stats** - games played, games won, best tile, total moves, total score.
- **Light / Dark theme** applied live across the whole UI.
- **Procedural audio** - move/merge/win/lose SFX synthesized at runtime (just for fun)
- **Haptics** 

---


### Controls

- **Mobile:** swipe up / down / left / right anywhere over the board.
- **Editor:** mouse drag over the board, or arrow keys / WASD.
- **MENU** opens settings & stats; **UNDO** reverts the last move; **NEW GAME** restarts.
